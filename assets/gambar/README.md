# Folder Gambar

Letakkan foto profil Anda di sini dengan nama `foto_profil.png`

## Format yang Didukung
- PNG (disarankan)
- JPG/JPEG
- WEBP

## Ukuran yang Disarankan
- Minimal: 500x500 px
- Format: Square (1:1)
- File size: < 2MB untuk performa optimal

## Cara Menambahkan Foto Profil

1. Siapkan foto Anda
2. Rename menjadi `foto_profil.png`
3. Copy ke folder `assets/gambar/`
4. Jalankan `flutter pub get`
5. Restart aplikasi (hot restart)

---

**Note:** Jika foto tidak ditambahkan, aplikasi akan menampilkan icon person default.
